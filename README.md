# MalazNganggur - Desktop ATS CV Generator

**MalazNganggur** adalah aplikasi desktop untuk membuat **CV ATS-friendly** secara cepat dan mudah.  
Cukup masukkan data pribadi, pengalaman kerja, pendidikan, dan skill, lalu aplikasi akan generate CV dalam format yang mudah dibaca oleh sistem rekrutmen otomatis (ATS).
